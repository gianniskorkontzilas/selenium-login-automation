# Selenium Login Automation Test

This project contains an automated login test using:

- Java
- Selenium WebDriver
- WebDriverManager
- Maven

## Test Scenario

The test performs:

1. Navigate to https://the-internet.herokuapp.com/login
2. Enter username: tomsmith
3. Enter password: SuperSecretPassword!
4. Click Login button

## How to Run

Clone the project:

git clone https://github.com/gianniskorkontzilas/selenium-login-automation.git

Open with IntelliJ and run LoginTest.java
